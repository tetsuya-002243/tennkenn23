橋梁点検 写真野帳 v21 PDF Fix

修正内容:
- PDFを追加できない根本原因を修正
- v20ではPDF本体をlocalStorageへ保存していたため、PDF容量が大きいと保存容量を超えて追加に失敗する可能性がありました
- v21ではPDF本体をIndexedDBに保存し、localStorageには橋梁名・PDF名・写真アイコン情報などの軽い情報だけを保存します
- 複数橋梁・複数PDF管理は維持
- PDF追加後、自動で編集画面へ移動してPDFを開く処理を維持
- PDF本体が見つからない場合は、再追加を促すエラーを表示します

注意:
iPad/Safariで古い版のキャッシュが残っている場合は、GitHub Pagesを再読み込みしてください。
ホーム画面に追加済みの場合は、一度ホーム画面アイコンを削除して、再度追加した方が確実です。
